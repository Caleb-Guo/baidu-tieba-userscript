# 修改
* 无用户名的贴吧账号无法正常运行此脚本
* 修改为只在各个贴吧的主页和主题贴里运行
* 修改了屏蔽显示样式，已避免特殊情况下，导致楼层错位（'position: absolute;'改为'position: relative;''）,加深了屏蔽颜色
* 发主题贴或回贴后，屏蔽样式可能会消失，刷新贴吧即可
* 发贴后可能会误判断，刷新一下贴吧应该可以解决
